# Collaborative Studio

A real-time collaborative application built with Flask and SocketIO.

## 🚀 Features
- Real-time communication using SocketIO.
- Lightweight and easy to deploy.
- Persistent data storage using SQLite.

## 🛠️ Tech Stack
- **Backend:** Flask
- **Communication:** Flask-SocketIO
- **Database:** SQLite
- **Deployment:** Render

## 💻 How to Run Locally

1. **Clone the repository:**
   ```bash
   git clone [https://github.com/Maahijoshi28/collaborative-studio.git](https://github.com/Maahijoshi28/collaborative-studio.git)
   cd collaborative-studio
Install dependencies:

Bash
pip install -r requirements.txt
Run the application:

Bash
python app.py
Open your browser:
Visit http://127.0.0.1:5000

🌐 Deployment
This project is automatically deployed on Render.
